# Biblioteca Pessoal em Python

Projeto simples desenvolvido em Python para gerenciamento de livros, utilizando conceitos básicos de programação e estruturas de dados.

## Objetivo do Projeto

O sistema permite cadastrar livros, listar os livros cadastrados, atualizar o status de leitura e visualizar o histórico de livros concluídos.

## Funcionalidades

- Cadastro de livros
- Listagem de livros
- Atualização de status
- Histórico de livros concluídos

## Estruturas de Dados Utilizadas

O projeto utiliza:

- Listas
- Filas (FIFO)
- Pilhas (LIFO)
- Tuplas
- Dicionários

## Organização dos Arquivos

### main.py
Arquivo principal do sistema, responsável pelo menu e navegação.

### tarefas.py
Contém as regras de negócio do sistema:
- cadastro
- listagem
- atualização de status
- histórico

### dados.py
Armazena os dados globais do projeto:
- listas
- filas
- pilhas
- tuplas

### utils.py
Contém funções auxiliares, como títulos e organização visual do terminal.

## Como Executar

Abra o terminal na pasta do projeto e execute:

```bash
python main.py
```
